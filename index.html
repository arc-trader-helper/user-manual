What Arc Trader Helper does

Main use cases:

Offers: create / edit / refresh / delete
Alerts: item notifications
Seeds: trades for seeds
Barter: item barter
Goal: less manual routine and fewer missed profitable marketplace offers.

Below is a step-by-step guide with screenshots.

1) Creating an offer

Screenshots 1–2 show the process of creating a new listing.

Simple flow: first choose the offer type, then the item and trade conditions.
  1. Click Create a new listing.
  2. Choose the offer type:
    Seeds — trading for seeds
    Barter — item barter
    Offer — offer/listing
  3. Select the item (what you want to buy/sell) and set the quantity.
  4. Choose the trade type: BUY or SELL.
  5. Fill in the conditions depending on the type:
    • Seeds: set the amount of seeds.
    • Barter: choose the item(s) and their quantities.
  6. Add a description (optional) and click Create.

2) Managing offers ( / refresh / edit / delete)

Screenshots 3–4 show where your active offers are and what actions are available.

Delete — delete the offer
Refresh — republish / bump the offer
Edit — edit the offer details
  1. Open the Offers tab and select status Active.
  2. Find the offer in the list and choose an action:
    • Delete — deletes the offer.
    • Refresh — refreshes (bumps) the offer to make it current.
    • Edit — opens the edit form.

✏ Editing an offer (example: Seeds)

This example uses Seeds. For Barter and Offer the flow is similar — only the condition fields differ.

  1. Click Edit.
  2. Select type: Seeds / Barter / Offer.
  3. Update the fields:
    • item quantity,
    • trade type BUY or SELL,
    • seed amount (for Seeds),
    • description.
  4. Click Update.

3) Reactivating an offer (Reactivate)

If an offer was completed, canceled, expired, or deleted, you can restore it using Reactivate. After reactivation, the offer will appear again in Active.

  1. Open the Offers tab.
  2. In the Status dropdown, select one of:
    Completed — completed
    Canceled — canceled
    Expired — expired
    Deleted — deleted
  3. Find the offer you want to restore.
  4. Click Reactivate.
  5. Check the Active — the offer will appear there again.

4) Alerts — item

Screenshots 5–6 show how to create an alert: when a matching offer appears, you will get a notification.

  1. Go to Alerts.
  2. Click Add alert / Create alert.
  3. Choose tracking type:
    Any — any offers
    Seeds — for seeds
    Barter — barter
    Offer — offers
  4. Select the item you want to track.
  5. Set conditions (example Seeds):
    • item quantity,
    • set the maximum seeds you are willing to pay.
  6. Click Create. When a matching offer appears on the marketplace, you will receive a notification 🔔

Tips

  • Use Refresh, to keep your offers visible and up-to-date.
  • For scarce items, create Alerts — this reduces the chance of missing a good price.
  • In the offer description, add important conditions.
